ELI12

Bits are 0 or 1. Eight bits make a byte. Each bit spot has a value: 128 64 32 16 8 4 2 1 (left→right). If a bit shows 1, you add its value. That total is the decimal number.
Binary → Decimal
Add the values under the 1s. Example: 11001010 → 128+64+8+2 = 202.
Decimal → Binary
Start at 128 and go down. Use a bit if it fits. Example: 202 → fits 128 → remainder 74 → fits 64 → remainder 10 → fits 8 → remainder 2 → fits 2 → remainder 0 → 11001010.
Hex
1 hex digit = 4 bits. Group 8 bits into two groups of 4. Example: 11000000 = 1100 0000 = C0 = C0 = 192.
IPv4
An IP has 4 bytes. Example: 11000000.10101000.00000000.00000001 → 192.168.0.1 → hex C0.A8.00.01.
Masks
Inside one byte: /25 128, /26 192, /27 224, /28 240, /29 248, /30 252, /31 254, /32 255.
