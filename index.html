ELI12

Bits are 0 or 1. Eight bits make a byte. Each bit spot has a value: 128 64 32 16 8 4 2 1 (left to right). If a bit shows 1, add its value to get the decimal number.
Binary -> Decimal
Add the values under the 1s. Example: 11001010 -> 128 + 64 + 8 + 2 = 202.
Decimal -> Binary
Start at 128 and work down. Use a bit if it fits. Example: 202 -> fits 128 -> remainder 74 -> fits 64 -> remainder 10 -> fits 8 -> remainder 2 -> fits 2 -> remainder 0 -> 11001010.
Hex
One hex digit equals four bits. Group eight bits into two chunks of four. Example: 11000000 -> 1100 0000 -> C0 -> 192.
IPv4
An IP has four bytes. Example: 11000000.10101000.00000000.00000001 -> 192.168.0.1.
Masks
Inside one byte: /25 128, /26 192, ... , /32 255.
